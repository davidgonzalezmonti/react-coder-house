### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/davidgonzalezmonti/react-coder-house/tree/proyectoFinal
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the APP
   ```js
   npm start;
   ```
